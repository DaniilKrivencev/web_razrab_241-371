# Лабораторная работа №1 — Flask Blog

Персональный блог на **Flask** с генерацией случайных постов, комментариями и автотестами.

## 🛠 Стек

- **Backend:** Python 3, Flask 3.0
- **Шаблоны:** Jinja2
- **Стили:** Vanilla CSS (тёмная тема, Inter + Playfair Display)
- **Тесты:** pytest + pytest-mock

## 🗂 Структура

```
Лабораторная 1/
├── requirements.txt     # Зависимости
├── run.bat              # Быстрый запуск (Windows)
└── app/
    ├── app.py           # Маршруты Flask
    ├── __init__.py      # Пакет (нужен для тестов)
    ├── static/
    │   ├── styles.css   # Все стили
    │   └── images/      # Изображения постов и аватар
    ├── templates/
    │   ├── base.html    # Базовый шаблон (navbar, footer)
    │   ├── posts.html   # Список постов
    │   ├── post.html    # Страница поста
    │   └── about.html   # Об авторе
    └── tests/
        ├── conftest.py  # Фикстуры pytest
        └── test_posts.py# 19 тест-кейсов
```

## 🚀 Запуск

### Вариант 1 — через bat-скрипт
```
run.bat
```

### Вариант 2 — вручную
```bash
pip install -r requirements.txt
python app/app.py
```

Приложение откроется на **http://127.0.0.1:5000**

> ⚠️ Не открывай `.html`-файлы напрямую в браузере — Jinja2-шаблоны работают только через Flask-сервер.

## 🧪 Тесты

```bash
# Из директории "Лабораторная 1":
python -m pytest app/tests/ -v
```

## 📄 Маршруты

| URL | Описание |
|-----|----------|
| `/` | Главная страница (список постов) |
| `/posts` | Все посты |
| `/posts/<index>` | Конкретный пост (404 если нет) |
| `/about` | Об авторе |
